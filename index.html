Bag & Board

A single-page React app built to keep track of a comic book reader's collection, wish lists, and books to trade with friends. The app is designed with Bootstrap and utilizes a JSON server for fetches. Version 2.0 will utilize the Goodreads API so users will be able to search for collected editions and single issues from all publishers.

local_library

G.R.E.A.T.

A Guided Reading Elementary Assessment Tracker. This is a Python/Django Web app that allows elementary school teachers to track their students' reading accuracy & levels and skills to guide instruction with data. For this app, I chose to have a client to simulate real-world workflow regarding planning, testing, and meeting deadlines.
